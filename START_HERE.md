This is the root launcher file for the New Earth website project.

Open this file first whenever you return to the build.

It tells you:
- what this project is
- what the source of truth is
- what file to open next
- what order to work in
- how not to lose the structure

## WHAT THIS PROJECT IS

This project is the build workspace for:

**www.new-earth.uk**

It is the structured source of truth for the New Earth website and its future expansion.

This build is based on:

- the locked New Earth Blueprint
- a local file structure
- Codex-compatible project rules
- WordPress on IONOS as the publishing layer

---

## THE MOST IMPORTANT RULE

**The local workspace is the source of truth.  
WordPress is the publishing layer.**

Do not let live WordPress edits become the only version of the project.

Always reflect meaningful changes back into the local structure.

## WHAT NOT TO DO

Do not:
- build random new pages outside the structure
- rewrite the Blueprint casually
- overload WordPress with plugins
- let live edits drift away from local files
- jump into subpages before core pages are stable
- redesign everything in the middle of implementation
- skip logging meaningful sessions
