# CATEGORIES

## Page
Journal Categories

## Status
Draft v1

## Purpose
This file defines the official category structure for the New Earth Journal.

The goal is to keep the Journal:
- organized
- searchable
- coherent
- aligned with the wider website architecture

These categories should be used consistently in WordPress.

---

## CORE CATEGORY RULE

Use a small number of meaningful categories.

Do not create too many categories too early.
A clean Journal structure is easier to maintain and easier for visitors to understand.

---

## OFFICIAL JOURNAL CATEGORIES

### 1. Blueprint
Use for:
- posts explaining the Blueprint
- posts expanding the 12 pillars
- posts interpreting the New Earth framework
- structural doctrine pieces

Examples:
- What the New Earth Blueprint Is
- The 12 Pillars Explained
- The New Earth Test in Practice

---

### 2. Manifesto
Use for:
- worldview pieces
- philosophical reflections
- moral direction
- civilizational reflection
- deeper tone-setting posts

Examples:
- Why a Different Foundation Matters
- Life Is Interconnected
- Beyond Extraction and Fragmentation

---

### 3. Systems
Use for:
- systems thinking articles
- design frameworks
- structural analysis
- practical architecture pieces
- doctrine-to-implementation posts

Examples:
- Why Systems Matter More Than Slogans

- Designing for Coherence

---

### 4. MicroGrow

## CATEGORY USAGE RULES

### Rule 1
Each post should usually have one primary category.

### Rule 2
Only assign multiple categories if genuinely necessary.

### Rule 3
Do not create new categories impulsively.

### Rule 4
If a new category is needed, add it here first.

For version 1, the category system succeeds if:

- it is consistent
