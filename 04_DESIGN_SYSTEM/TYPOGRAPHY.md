# TYPOGRAPHY

## Status
Typography System v1

## Purpose
This file defines the typography direction for the New Earth website.

The goal is to create a type system that feels:
- calm
- intelligent
- grounded
- spacious
- readable
- quietly elevated

Typography should support both:
- the philosophical depth of New Earth
- the structural clarity of the systems framework
- the practical credibility of MicroGrow

---

## CORE TYPOGRAPHY INTENT

The New Earth typography system should feel like:

- clarity with soul
- structure with warmth
- wisdom without heaviness
- refinement without elitism

It should not feel like:
- hard corporate SaaS branding
- futuristic techno styling
- decorative spirituality
- overly ornate editorial design
- cramped minimalist severity

---

## TYPE ROLES

Use two main type roles:

### 1. Display / Heading Typeface
Used for:
- page titles
- section headings
- quote bands
- key declarations
- major anchors

This typeface should carry:
- elegance
- presence
- thoughtfulness
- depth

Recommended direction:
- Playfair Display
or
- Cormorant

These work well because they feel human, reflective, and elevated without losing readability.

---

### 2. Body Typeface
Used for:
- paragraphs
- menus
- buttons
- lists
- interface copy
- microcopy

This typeface should carry:
- readability
- calm neutrality
- modern clarity
- clean spacing

Recommended direction:
- Inter
or
- Source Sans

These work well because they are highly readable and pair well with a more expressive heading face.

---

## PRIMARY RECOMMENDED PAIRING

### Headings
Playfair Display

### Body
Inter

This pairing gives:
- a spiritual and reflective tone in headings
- practical, modern readability in body text

It is the strongest balance for New Earth v1.

---

## SECONDARY OPTIONAL PAIRING

### Headings
Cormorant

### Body
Source Sans

This pairing feels slightly softer and more editorial.
Use only if you want a more poetic feel.

For New Earth, the first pairing is the safer default.

---

## HIERARCHY RULES

### Page title
Use for:
- main page hero heading

Style goal:
- strong
- spacious
- unhurried
- visually anchored

Should feel like a statement, not a slogan.

---

### Section heading
Use for:
- each major section

Style goal:
- clear hierarchy
- readable
- consistent
- calm but distinct

Section headings should feel like meaningful chapter markers.

---

### Subheading
Use for:
- supporting structure under section headings
- feature headings
- card titles

Style goal:
- clean
- slightly firmer
- easy to scan

---

### Body paragraph
Use for:
- all core readable copy

Style goal:
- generous line spacing
- moderate line length
- no visual crowding
- natural reading flow

The body text carries most of the emotional and structural clarity of the site.
Do not compress it.

---

### Small text / interface text
Use for:
- labels
- menu items
- metadata
- supporting notes
- footer content

Style goal:
- legible
- quiet
- not overly stylized

---

## TYPOGRAPHIC FEEL BY PAGE

### Home
Should feel welcoming, spacious, and orienting.

### Manifesto
Should feel more reflective and elevated, while still readable.

### Blueprint
Should feel structured, serious, and clear.

### Systems
Should feel practical and architectural.

### MicroGrow
Should feel slightly more technical, but still within the New Earth tone.

### About
Should feel human and sincere.

### Journal
Should feel thoughtful and editorial.

---

## SPACING RULES

Typography is not just font choice.
It is also spacing.

### Rule 1
Use generous spacing between headings and paragraphs.

### Rule 2
Do not stack large text too tightly.

### Rule 3
Keep paragraph widths readable.
Avoid very wide text lines.

### Rule 4
Let declarations breathe.
Short powerful lines need space around them.

### Rule 5
Do not overuse bold text.
Use hierarchy, not force.

---

## BUTTON TEXT RULES

Button text should use the body font.

Reason:
Buttons should feel clear and functional, not ornamental.

Button text should be:
- short
- direct
- calm

Examples:
- Explore the Blueprint
- Discover MicroGrow
- Read the Manifesto
- Get in touch

---

## MENU TYPOGRAPHY RULES

Menu text should use the body font, not the heading font.

Reason:
Navigation must be easy to scan and visually stable.

Menu text should feel:
- clean
- quiet
- balanced
- unobtrusive

---

## QUOTE AND DECLARATION TYPOGRAPHY

Use heading type for:
- major declarations
- manifesto pull quotes
- closing lines
- key philosophical statements

Examples:
- Life is interconnected.
- Love becomes structure.
- We choose New Earth.

These lines should feel memorable without becoming theatrical.

---

## MICROGROW TYPOGRAPHY RELATIONSHIP

MicroGrow should use the same type system as New Earth.

Do not switch into generic tech branding fonts just to make it feel more technical.

MicroGrow should feel:
- structurally aligned
- slightly firmer
- slightly more functional
- still fully inside the New Earth visual world

---

## TYPOGRAPHIC ACCESSIBILITY RULE

Before finalizing typography, confirm:
- body text is easy to read on mobile
- headings are clearly distinct from paragraphs
- buttons remain readable
- line spacing is comfortable
- no section feels crowded

Beauty must support reading.
