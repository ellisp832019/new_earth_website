# PAGE_HIERARCHY

## Status
Page Hierarchy v1

## Purpose
Defines the structural relationships between all pages on the New Earth website.

This ensures:
- clarity
- scalability
- logical navigation
- alignment with the Blueprint

## 1. Hierarchy model

The site is structured as a layered system:

### Level 0 — Root
- Home

### Level 1 — Core Pages
- Manifesto
- Blueprint
- Systems
- MicroGrow
- Journal
- About
- Contact
- Legal
