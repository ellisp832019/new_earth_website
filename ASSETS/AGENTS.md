# AGENTS.md

## Scope
This folder contains the locked visual asset production system for the New Earth website and related MicroGrow pages.

All work inside this folder must preserve:
- folder structure
- agreed filenames
- prompt batch files
- `ASSET_REGISTER.csv` as source of truth

## Source of truth inside this folder
Use this priority:
1. direct user instruction
2. `../TASK.md`
3. `ASSET_REGISTER.csv`
4. `_PROMPTS/*.md`

## Rules
- do not invent extra batches
- do not invent extra sections
- do not rename agreed assets casually
- do not replace prompt wording unless asked
- do not add documentation bloat
- do not create unnecessary nested folders

## Status values
Only use:
- `TODO`
- `IN_PROGRESS`
- `DONE`
- `NEEDS_EDIT`
- `APPROVED`
- `LIVE`

## Expected docs
These files are allowed and useful:
- `ASSET_PRODUCTION_CHECKLIST.md`
- `ASSET_REVIEW_STANDARD.md`
- `ASSET_GAP_REPORT.md`

Keep them short and operational.

## Format defaults
- `.webp` for heroes, banners, and section images
- `.png` for diagrams, icon sheets, templates, posters, and graphics needing crisp edges

## Working style
Make small, safe, reviewable changes.
Treat this asset system as locked unless the user explicitly changes it.
