# Asset Production Checklist

Use this checklist to move assets from prompt to approved file without drifting from the current system.

## Production flow
1. Generate from the correct batch prompt file.
2. Review against `ASSET_REVIEW_STANDARD.md`.
3. Place the file in the correct `ASSETS/` folder.
4. Update `ASSET_REGISTER.csv`.
5. Mark the asset `APPROVED` when it passes review.
6. Export for web or print if needed.
