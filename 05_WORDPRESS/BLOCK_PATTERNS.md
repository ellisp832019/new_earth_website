# BLOCK_PATTERNS

## Status
Block Patterns v1

## Purpose
This file defines the reusable WordPress content patterns for the New Earth website.

The goal is to create:
- consistency
- speed
- clarity
- scalability
- design coherence

These patterns should be reused across pages rather than rebuilt from scratch every time.

---

## CORE RULE

Build reusable patterns for repeated section types.

Do not design every page from zero.

Patterns should support:
- visual calm
- strong hierarchy
- clean spacing
- doctrine clarity
- flexible reuse

## PATTERN DESIGN RULES

### 1. Reusable first
If a section type repeats, make it a reusable pattern.

### 2. Calm hierarchy
Patterns should create rhythm, not noise.

### 3. Content leads
The pattern should support the message, not overpower it.

### 4. Keep editing simple
Patterns should remain easy to update in WordPress.

### 5. Avoid over-decoration
New Earth should feel intentional and clear, not busy.

---

## FINAL PATTERN TEST

Before creating a new custom section, ask:

1. Does an existing pattern already solve this?
2. Can this be simplified into a reusable block?
3. Will this pattern still make sense across multiple pages?
4. Does it support the New Earth visual tone?

If yes, build it as a reusable pattern.
