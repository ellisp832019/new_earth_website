# REUSABLE_SECTIONS

## Status
Reusable Sections v1

## Purpose
This file defines the reusable section library for the New Earth website.

The goal is to make page building:
- faster
- more consistent
- easier to maintain
- more coherent across the whole site

These reusable sections should become your standard WordPress build blocks or patterns.

---

## CORE RULE

If a section type appears more than once, it should become reusable.

Do not rebuild common sections from scratch every time.

## WORDPRESS BUILD RULE

When building pages in WordPress:

1. create reusable patterns for common sections
2. keep section names clear
3. use the same spacing and structure each time
4. adjust content, not the core pattern logic

This makes the site easier to scale and easier to maintain.

---

## SECTION NAMING RULE

Use clean names in WordPress for reusable patterns, such as:

- NE Hero
- NE Text Intro
- NE Three Card Block
- NE Pillar Grid
- NE Declaration Band
- NE Two Column Bridge
- NE Feature List
- NE Timeline
- NE Journal Preview
- NE Closing CTA

This keeps your pattern library organized.

---

## FINAL REUSE TEST

Before creating a custom one-off section, ask:

1. Is this already covered by an existing reusable section?
2. Can this be simplified into a reusable structure?
3. Will this likely appear on another page?
4. Does reuse improve consistency here?

If yes, build it as reusable.
